== Description ==
This plugin allows WooCommerce store owners to set a custom currency symbol. User can add any unicode text or text symbol.

Go to WooCommerce Settings. Click General settings.
Now there is a text box to enter the currency symbol.
Enter the custom currency symbol.
Then Save and enjoy. Simple plugging.


This plugging not affacts to your website loading speed
You can add any unicode text. And its appear as your currency symbol

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/custom-currency-sign` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to WooCommerce > Settings > General and enter your custom currency symbol.
4. Save
